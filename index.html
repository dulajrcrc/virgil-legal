Virgil: Prediction Markets

Privacy Policy

Last updated: June 2026

1. Information We Collect

Virgil collects only what is strictly necessary to provide the service:

  • Wallet addresses you voluntarily provide (Polymarket) — publicly visible blockchain data
  • API key identifiers you provide (Kalshi) — stored with AES-256 encryption, never transmitted in plain text
  • Anonymous usage analytics (screen views, feature interactions) — no personally identifiable information
  • Push notification tokens if you enable notifications
  • Device type and OS version for compatibility purposes

WE DO NOT COLLECT: your name, email, payment information, location data, contacts, or photos.

2. How We Use Your Information

We use collected information exclusively to display your prediction market positions, calculate analytics, send opted-in push notifications, and improve app performance. We will never sell your data, use it for advertising, or share credentials with third parties.

3. Data Security

Kalshi API keys are encrypted at rest using AES-256. All data in transit is protected by TLS 1.3. API keys are never logged or displayed in plain text. In the event of a breach we will notify affected users within 72 hours.

4. Third-Party Services

Virgil connects to Polymarket, Kalshi, and an AI analysis service on your behalf. Each operates under its own privacy policy. We are not responsible for third-party data practices.

5. Data Retention

Wallet addresses and API keys are stored locally on your device only and deleted when you disconnect. You can delete all data at any time by disconnecting accounts in Settings and uninstalling the app.

6. Children's Privacy

Virgil is not directed at children under 13. We do not knowingly collect data from children under 13.

7. Changes to This Policy

We may update this policy from time to time. We will notify users of significant changes via the app.
